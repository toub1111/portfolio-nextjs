# Portfolio Imad Nagifi

Un portfolio moderne et interactif développé avec Next.js et Redux, présentant un design Neo-Skeuomorphism minimaliste.

## 🚀 Caractéristiques

- **Design Neo-Skeuomorphism** inspiré des interfaces 3D avec des ombres douces et des contrastes subtils
- **Authentification complète** (login/register) gérée par Redux
- **Protection des routes** pour les utilisateurs non authentifiés
- **Système de témoignages** avec possibilité d'ajouter/modifier pour les utilisateurs connectés
- **Pages de projets détaillées** présentant les compétences et réalisations

## 🛠️ Technologies utilisées

- **Frontend**: Next.js, Redux Toolkit
- **Design**: Neo-Skeuomorphism, interface utilisateur minimaliste

## 🚀 Installation

```bash
npm install
npm run dev
```
